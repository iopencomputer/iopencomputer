# Rust EBNF (Reference)

This project uses the official Rust Reference grammar as the source of truth.

The canonical grammar summary is published in the Rust Reference under the
"Grammar summary" section. It is maintained by the Rust project and updated
with each release.

Source:
- https://doc.rust-lang.org/reference/grammar.html

Notes:
- The Rust Reference uses its own EBNF-like notation (see the Notation section).
- The grammar is very large and updated every release, so we link to the
  canonical source instead of embedding a static copy here.

Notation reference:
- https://doc.rust-lang.org/reference/notation.html
