# iOpenComputer 計畫

> 設計一台電腦，包含軟體和硬體

> 重建整個軟體工業文明

> 本計劃由 AI 輔助設計

本計劃選定使用 rust 程式語言，實作下列子計畫軟體

1. iCompiler -- 編譯器，中間碼用 LLVM IR，最後轉為 iCPU 可以執行的機器碼。
2. iOSiX -- 作業系統，必須可以在 iCPU 處理器上執行。
3. iCPU -- 處理器，相容於 iMac M3 晶片
4. iBrowser -- 瀏覽器，支援 HTML/CSS/JavaScript 標準
5. iEDA -- 電子設計工具，支援 iHDL，是融合 rust 與 Verilog 的硬體的描述語言。
